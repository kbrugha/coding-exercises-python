# coding-exercises-python

## Usage


### FizzBuzz

#### Using defaults

    mod_list = {3: "Three", 5: 'Five'}
    length = 100

`fizzbuzz.py`

#### Using alternative options
    mod_list = {3: "Fizz", 5: 'Buzz'}
    length = 15

`fizzbuzz.py --length=15 --mod 3="Fizz" 5="Buzz"`

### Postcode Validation
`postcodes.py --validate "AB10 1XG" "ab10 1xg"`

### Postcode Formatting
`postcodes.py --format "ab101xg"`
